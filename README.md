# Insurance Reminder System

A modern **Insurance Reminder System** built using **Spring Boot, Spring MVC, Thymeleaf, PostgreSQL, HTML, CSS, and JavaScript**. This application helps users manage insurance policies efficiently by sending timely reminders before policy expiration.

---

## 📌 Features

* 🔐 User Registration & Login
* 🔒 Secure Authentication
* ➕ Add New Insurance Policies
* ✏️ Edit Existing Policies
* ❌ Delete Policies
* 📋 Dashboard with Policy Overview
* ⏰ Expiry Reminder System
* 📧 Email Notifications for Expiring Policies
* 📱 Responsive UI
* 🗄 PostgreSQL Database Integration

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Insurance_Reminder_System.git
```

2. Open the project in IntelliJ IDEA.

3. Create a PostgreSQL database.

4. Update the database configuration in:

```properties
application.properties
```

5. Configure your email credentials.

6. Run

```bash
mvn spring-boot:run
```

7. Open

```
http://localhost:8080
```
